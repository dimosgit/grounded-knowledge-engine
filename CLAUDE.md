Claude Code specifics:

- This repo's own `kb` MCP server is registered via a machine-local
  `.mcp.json` (generated by `npm run setup:mcp`, gitignored) plus local
  approval, so `mcp__kb__*` tools (search, get_record, answer_and_capture,
  resume_project) may be available in-session. Treat them as the product under
  test: prefer reading source files directly when debugging the server itself.
