# Unknown Procedure Abstention Boundary

Zephyr-Null is a deliberately nonexistent synthetic procedure. If a question
asks whether a missing Zephyr-Null procedure can be inferred from a similar
project, the grounded response must abstain and request evidence. Similar names
are not permission to invent a procedure.
